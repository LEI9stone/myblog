## 食材

+ 黄牛肉（一半腿肉，一半肋条），提前烀熟的黄牛肉切成片
+ 豆芽

## 调味
+ 食用油
+ 葱适量
+ 姜适量
+ 八角一个
+ 老抽
+ 生抽
+ 鸡精
+ 味精
+ 盐
+ 胡椒粉

## 烹饪

先炒豆芽，把豆芽水分炒干，小火慢慢烧，不糊就行。豆芽皮炒掉，豆芽微微发焦后，盛出备用。
再次烧油，葱姜少许、八角一个，掰碎；炸出香气后，放入牛肉。牛肉炒的稍微起焦，放入豆芽混炒。
放入老抽上色，大火炒出酱香，生抽提鲜，放入适量水，小火慢炖。在小火慢炖的过程中，放入鸡精、味精、盐、胡椒粉调味。
