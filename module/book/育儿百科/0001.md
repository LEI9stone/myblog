# 分娩和分娩后

在身体和其他各方面条件允许的前提下，建议选择自然分娩。其实在医院里，医生会评估相关情况，医生也会建议选择自然分娩。孕妇一般会在妊娠第37 ~ 42周分娩。

## 分娩后

建议选择母乳喂养，新生儿与宝妈进行肌肤接触后的1小时内进行母乳。最好在宝宝出生后1小时内和宝妈有肌肤接触，这样有助于宝宝和宝妈的互相了解，还有益健康。分娩后立即开始哺乳对宝妈有益，可促进子宫收缩（负责引发射乳反射的激素也会刺激子宫收缩），减少产后出血，还能提高新生儿的免疫力。

分娩后1小时左右是开始母乳喂养的 “黄金时间”，因为这时新生儿的反应机敏，正是想吃奶的时候。需要让宝宝将乳晕连同乳头含在嘴里用力吮吸，而非仅仅含住乳头。如果1个小时以后开始哺乳，宝宝很可能已经困了，很难正确吮吸。

:::tip 分娩后的哺乳
新生儿出生后1 ~ 3天，母亲的身体会分泌初乳。初乳是一种淡黄色的液体，量很少，但含有丰富的蛋白质和抗体，可以保护新生儿免受感染。初乳可满足新生儿出生后前几日的所有养分和水分需求。
:::


即使选择刨宫产手术，只要孩子健康状况稳定，也鼓励立即进行肌肤接触以建立亲密关系，也鼓励进行母乳喂养。

:::tip 分娩后立即接触的必要性
母亲和孩子若均健康且状况稳定，应在分娩后的最初几小时内，尽可能多地增加肌肤接触的机会。肌肤接触有助于稳定新生儿的血糖水平、血液循环、呼吸以及体温，还有助于预防低温症（体温过低）、减少啼哭，这对晚期早产儿尤其有益。对母亲来说，肌肤接触可以减轻分娩后的精神压力，并可能减少产后出血。
:::

+ 胎脂：大多数新生儿刚出生时身上裹着一层白色乳酪状物质。

## 脐带血

脐带血可以成功治愈大量遗传病、血液病及儿童癌症，如白血病及免疫系统疾病等。

:::tip 脐带血
脐带血是孩子诞生、脐带被夹住并切断之后，从被切断的脐带中采集的，这个过程不会对新生儿或分娩过程产生不利影响。脐带血的采集并不影响脐带处理的常规流程。
:::
