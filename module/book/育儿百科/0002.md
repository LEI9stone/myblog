# 新生儿的最初几日

**敏感型宝宝**。从出生第一天起就对尿湿或拉脏的尿布表现出零容忍的态度，他们会用大哭大闹来表示抗议，直到有人帮他们换掉脏尿布，把他们喂饱，然后轻轻摇晃着把他们哄睡。通常，这类敏感型新生儿不仅其他新生儿清醒的时间更长，而且哭闹和吃奶的次数也更多。

**非敏感型宝宝**。另一类新生儿似乎根本不会留意到尿布脏了，反而可能抗议换尿布时屁股暴露在外的感觉。这类新生儿通常睡得多，吃奶频率不像敏感型新生儿那么高。

这两类宝宝的表现都是正常的，我们从中可以看出孩子的性格特质。

::: tip
尊重孩子的个性，是为人父母的重要职责。父母认可孩子的独特性，日后才可以更轻松地接受长大成人的孩子。
:::

### 了解新生儿

- **眼白有血点**：眼白上的血点和新生儿面部水肿一样，通常都是因分娩过程中受到产道挤压造成的。不用担心，过几天会消失。刨宫产的新生儿出生时可能就没有面部肿胀或眼白出现血点的问题。
- 无论是早产儿、晚产儿还是足月儿，出现脱皮都是正常的，无须治疗。
- **细小的毛发**：肩膀和后背发现一些细小的毛发，这些是胎毛。胎毛在孕晚期出现，通常新生儿出生前后出生后不久脱落。
- 如果发现脸上有抓痕，尤其是脸上。需要给新生儿修剪指甲。及时修剪指甲可以防止新生儿的手乱动时抓伤自己。

::: details 新生儿皮肤上常见的印记

- **单纯性痣（旧称“鲑鱼斑”或“鹳咬痕”）。** “鹳咬痕”的说法源于西方“白鹤送子”的传说，这种色斑常见于传说中婴儿被鹳嘴衔着的部位，因此得名。鲑鱼斑通常呈块状，颜色为粉色，但有深有浅，最常见于新生儿的鼻梁、额头下部、上眼皮、脑后和（或）颈部。这是一种常见的胎记，在浅肤色人种的婴儿身上尤为多见，也被称为“天使之吻”。它通常在几个月到几年内消失，但消失后也可能因脸红而暂时显现。
- **先天性真皮黑素细胞增多症（蒙古斑）**。这种胎记有大有小，源于皮肤中黑色素细胞增多，呈棕色、灰色甚至蓝色（像淤青），最常见于腰部或臀部。这种胎记也很常见，尤其是在深肤色人种的婴儿身上，通常会在学龄前消失，无需治疗。
- **新生儿暂时性脓包性黑变病**。这是一种小疱疹，通常在新生儿出生时出现，几天内会破掉，然后结痂，留下类似于雀斑的黑色斑点，斑点几周后会消失。有些新生儿出生时就带有这类黑斑，这说明他们在出生前就已经出现过这种疱疹。虽然脓包性黑变病比较常见（尤其是在深肤色人种的新生儿身上），属于无害的新生儿皮疹，但父母仍需请医生对所有疱状性皮疹进行检查，以便排除感染因素。
- **粟丘疹**。这是一种由皮脂腺分必物堆积形成的微小的白色突起或黄点，多分布在脸颊，下巴或整个鼻头。这种常见的新生儿皮疹通常在出生后2~3周之内自行消失。
- **粟粒疹**。俗称“痱子”，又称“热痱”或“汗疹”。痱子常出现在炎热而潮湿的天气里，但也可能因婴儿被包裹得太严实而出现。这种皮疹可能带有细小的汗疱和（或）小红点，最常见于皮肤褶皱处和被包裹得部位，通常几天内消失。
- **新生儿中毒性红斑**。这种皮疹虽看起来令人担忧，但很常见，一般在新生儿出生后几天内出现。中毒性红斑表现为皮肤出现多个红色斑块，红斑中间有乳黄色的小突起，1周左右完全消失。
- **血管瘤**。血管瘤是由皮肤中大量交织的血管形成的隆起的红斑。在新生儿出生后1周左右，血管瘤可能呈白色或灰色，之后会变成红色并隆起。血管瘤通常会在孩子1岁前变大，但大多数会在孩子达到入学年龄自行萎缩并消失。根据血管瘤的位置和大小，儿科医生或皮肤科医生也可能建议对其进行治疗。
- **葡萄酒色斑**。葡萄酒色斑是由皮下血管畸形造成的，通常分布在面部或颈部，表现为大块、平滑、不规则的深红色或紫色斑块，与血管瘤不同的是，这种胎记未经治疗不会自行消失。葡萄酒色斑有时可以通过激光手术治愈。

:::

- 卤门：未成熟的颅骨继续生长的空间。较大的卤门位于头顶较靠前的位置，较小的卤门位于头后部。可以轻轻抚摸，父母无须太紧张。但不能按压。
- 胎发：数量、发质、发色存在个体差异，大部分或全部的胎发会在6个月以内脱落。

### 新生儿体重

孕40周出生的新生儿（足月儿），80%的体重为2.6 ~ 3.8kg，这是健康新生儿的正常体重范围。体重大于3.8kg属于偏大的新生儿，体重小于2.6kg属于偏小的新生儿。

偏大的新生儿可能会出现代谢异常（比如低血糖、低钙血症）、产伤、血红蛋白含量高、黄疸或各种出生缺陷。儿科医生会关注这些问题，父母无须多度焦虑。

### 早产儿

早产：指胎儿在孕妇妊娠满27周之前出生，可分为晚期早产（第34~36周出生）、中期早产（第32~33周出生）、早期早产（第29~31周）出生和超早期早产（不满28周出生）。

早产儿因为出生时各器官系统尚未完全发育成熟，通常需要在新生儿重症监护室或特殊护理室接受一段时间的专门治疗和护理，特别是超早期早产儿。

只要医生允许，你就可以喂奶，母乳是最好的营养品，可提供抗体等物质来增强孩子的免疫力，帮助他抵抗感染。母乳对早产儿还有额外的益处：母乳有助于预防一种叫作“新生儿坏死性小肠结肠炎”的并发症。

分娩后，产妇应尽快开始挤奶，并保持2~3小时挤1次或每天挤8次的评率，这有利于乳汁分泌充足。如果选择母乳喂养，请坚持挤奶。

::: details 早产儿的健康问题

- **呼吸窘迫综合症**。婴儿肺部发育未成熟而引起的呼吸异常。早产儿的肺部通常缺乏一种叫做“肺表面活性物质”的化学物质。这种液体排布在微小的肺泡表面，能够在每次呼吸结束时维持肺泡张力。
- **支气管发育不良**。一种慢性肺病，患病的早产儿在出生几周甚至几个月后仍需要吸氧。随着早产儿的肺部逐渐发育，较晚出生的早产儿症状通常会逐渐好转。
- **呼吸暂停**。早产儿的常见问题，通常表现为呼吸突然暂停中断（超过20秒），通常伴有心跳减缓及血氧饱和度下降。好消息是大多数早产儿在出院时已经不存在呼吸暂停的问题了。
- **早产儿视网膜病变综合症**。由视网膜发育不完全导致的眼部疾病，会导致眼后部（视网膜）血管出血。大部分患儿无须治疗就会好转，但病情严重的患儿需要治疗，极少数需要进行激光手术或药物注射。长期并发症包括近视和失明。
- **脑室内出血**。常见于早产儿，尤其是出生体重不到1.5kg的早产儿，症状表现为脑室（大脑中含有脑脊液的部位）内或其周围出血。病因是早产儿的大脑内血管发育不成熟，极其脆弱，很容易破裂出血。大多数患有脑室内出血的早产儿会在出生后第一周内出现这种症状，可通过头部超声检查进行诊断。轻度出血（1级和2级）最为常见，通常不会出现并发症；重度出血（3级和4级）非常严重，可造成婴儿长期脑损伤。
- **脑性瘫痪**。早产儿，尤其是有脑室内出血或脑积水的患儿，存在患脑性瘫痪的风险。脑性瘫痪指大脑在控制运动、力量或姿势方面存在障碍，可以通过专项检查来筛查。对存在患脑性瘫痪风险的婴儿进行筛查至关重要，因为越早进行物理治疗，效果越好。
- **感染**。早产儿有受感染的分险。感染的症状与早产儿的症状相似，主要包括呼吸暂停、心动过缓（心率过低）、血氧饱和度下降（血氧不足）、需要呼吸机支持、氧气需求增加、腹胀、喂养不耐受和低血压等。如果出现了这些症状，婴儿可能需要进行败血症检查（包括血细胞计数、血液培养、尿液培养，有时还需要进行脊髓液培养）。婴儿可能需要接受坑生素治疗。如果48小时候所有培养结果呈阴性，则表明没有感染发生，抗生素就会被停用。如果任何培养结果呈阳性，医生将根据感染类型予以为期7~21天的抗生素治疗。
- **新生儿坏死性小肠结肠炎**。早产儿最常发生也是最严重的一种肠道疾病，多发于小肠（偶发于大肠）组织受到损害或发炎时。它会导致肠组织坏死，在极少数情况下，甚至会导致肠穿孔。新生儿坏死性小肠结肠炎多发于不满32周的早产儿，通常发病于出生后的第2~6周，症状为喂养不耐受或腹胀，可通过腹部超声或X线检查确诊。母乳喂养是预防新生儿坏死性小肠结肠炎最有效的方法之一。
- **黄疸**。这是红细胞自然分解导致一种名为“胆红素”的物质在婴儿血液中蓄积造成的，可使婴儿肤色发黄。
- **发育迟缓和残疾**。这种在所有儿童中都很常见（1/6的发病率），但在早产儿中更为普遍。“发育迟缓”指孩子在玩耍、学习、说话、行为和动作方面发育迟缓。早产儿在子宫内尚未发育完全，所以他们无法表现出与同龄足月新生儿同等的发育技能。因此，儿科医生经常使用早产儿的矫正年龄（即假设早产儿在预产期出生并据此计算其年龄）来判断其是否存在发育迟缓问题。家长和其他看护者对孩子的发育状况有任何忧虑，都应该及时向医生咨询。

:::

### 新生儿的行为

- 新生儿的视力范围在20 ~ 30厘米

出生起，新生儿可以分辨明暗，能够看到鲜艳的颜色，但还不能看到所有的颜色。给小婴儿看黑白图片或颜色对比强烈的图片，他会很有兴趣地仔细盯着。

- 研究显示，亲密的拥抱可以促进婴儿的生长发育

::: tip 出生几日需要注意的情况

- **面部或躯干呈青色**。手脚发青或发紫是正常现象，但如果面部或躯干变青，那可能表明婴儿的心脏或肺部出现了问题。
- **每分钟呼吸超过60次，或肋骨间（肋骨下）的皮肤被牵拉**。婴儿的呼吸通常会先快后慢再快，如果他的呼吸持续过快或过猛，请告知护士。有些婴儿在出生时呼吸状况良好，但在出生后的几小时内会出现呼吸问题。
- **一侧或两侧肢体的节律性异常运动**。新生儿可能会颤抖，即使是轻微的噪声或环境变化也会吓到他，这些都是正常的。但如果你注意到他的手臂或腿部出现持续性的、有节律的运动，这可能是抽搐的征兆。

:::

### 出院回家

- 出院需要做一系列的检查，比如听力筛查、遗传病和血筛查等
- 如果衔乳或母乳喂养时长方面出现问题，请与哺乳专家或儿科医生联系，并安排出院后的随访，以便更好地获得母乳喂养方面的帮助。
- 如果新生儿出现精神萎靡、发热、呕吐、无法进食或皮肤发黄（黄疸）等症状，父母应立即联系儿科医生。
- 带新生儿回家前，最好给汽车安装符合国标认证的安全座椅，尽量选择大仰角，且安全座椅朝向为反向。
- 需在家里准备好最基本的安全防护、尿垫、尿不湿、恒温壶、保暖的衣服和毯子。

### 家人的感受

#### 母亲的感受

你可能会发现自己在孩子出生后的最初几天里有一种快乐、痛苦和疲惫交织的感觉，你可能对自己为人母的能力有所怀疑，尤其是当这是你的第一个孩子时。

你可能因刚刚降生的小生命而异常兴奋，甚至忘记了自己身上疲惫和疼痛。虽然身体和精神都很疲惫，你却可能紧张得无法入睡。

你或伴侣在精疲力尽或昏昏欲睡的时候，一定要把孩子以仰卧的姿势放回摇篮里。

初为人母这一过度期容易令人焦虑，你需要一段时间才能习惯婴儿的日常护理工作，并让一切步入正轨，所以请给自己足够的宽容。请记住，你可以向信任的朋友、家人和儿科医生寻求帮助和建议。

**新生儿是否会影响你和其他孩子之间的感情？**

这种情况不一定会出现，只要你在安排日常生活时有意识地让其他孩子也参与进来。两三岁的孩子通常很乐意帮你给宝宝洗澡或者拿干净的尿布，再大点儿的孩子则会因帮你解决麻烦感到骄傲，比如他会帮你找丢失的玩具，督促每个客人在接触小宝宝前洗手等。你熟悉了新的日常生活后，要记得留出一些时间给其他孩子。

**你能给新生儿同等的爱？**

事实上，每个孩子都是独一无二的，会让你有不同的反应和感觉。甚至孩子出生的顺序也会影响你们的关系模式。要经常提醒自己“新”并不代表“更好”或“更不好”，通常只是有所不同而已。这是一个很重要的认知，你和孩子都要牢牢记住。有一个兄弟姐妹对孩子来说是一份礼物。孩子们能够在家庭环境中学到分享、安慰和妥协等社会情感能力。

你绝对不是世界上第一个哭鼻子的新手妈妈，当然也不会是最后一个。青春期的激素变化跟孕期与分娩后的激素波动比起来根本不值得一提。

情绪变化有时会令新手妈妈们感到难过、恐慌、易怒或焦虑，甚至对婴儿发脾气————医生称此为“产后沮丧”。

一些刚生产的妈妈会有更严重的悲伤、空虚、心灰意冷甚至绝望的感觉，这是产后抑郁的症状。她们还会生出极端的情绪，感到自己很没用，开始疏远家人和朋友。其症状可能会在分娩几周后出现。这些症状持续数月（甚至可能超过1年），病情会随着时间的推移而恶化，变得非常严重。即便是没有经历分娩的家长，同样可能患产后抑郁症，因此爸爸们也应对自身的产后抑郁迹象保持警惕，并尽早寻求帮助。

与伴侣、家人以及亲近的朋友讨论自己的感受。要允许他人帮忙照顾孩子。试着通过自我照顾来减轻压力和焦虑————睡眠、锻炼、冥想、与朋友交流，做任何能让你产生幸福感和活力的事情。假如情况仍然没有改善，父母安情绪仍然很严重，且2周内没有缓解，你就应该与产科医生或儿科医生谈谈，或者向精神科医生寻求帮助。

### 健康观察

**腹胀**。大多数婴儿的小肚子圆溜溜地鼓着，尤其是在他们饱餐一顿之后。然而，在两次喂奶之间，他们的肚子摸起来应该是柔软的。如果孩子的肚子摸起来又胀又硬，或者他已经一两天没有排便了，或者出现呕吐，你应该立即带他去看儿科医生。这种情况很有可能是胀气或排便困难造成的，也有可能是孩子患了严重的肠道疾病。

**产伤**。在分娩过程中，新生儿可能受伤，特别是当产程持续时间长，分娩比较困难或胎儿比较大的时候。虽然新生儿的产伤一般都可以很快恢复，但也有些新生儿的伤势会持续较长时间。锁骨骨折是一种偶尔会出现的产伤，很快就可以愈合。几周后，骨折处会出现一块小小的肿块，不要紧张，这是新骨正在形成、伤处愈合良好的征象，骨折处很快就会恢复如常。

肌无力是另一种可能出现的产伤，是由阵痛过程中胎儿支配相应肌肉的神经被压迫或多度牵拉造成的。受影响的肌肉常见于面部、肩部或上肢的一侧，一般几周后会自行恢复。你可以向儿科医生咨询如何护理以加速恢复。

**皮肤发青**。新生儿的双手和双脚可能微微发青，这是正常的。在较冷的环境中，新生儿的手脚会微微发青，等暖和起来，手脚就会恢复正常的粉红色。少数情况下，新生儿哭闹得太厉害时，他的面部、舌头以及臀部都有可能略微发青，但当他停止哭闹、安静下来时，发青的部位会恢复正常。然而，长期皮肤发青石心肺功能异常的迹象，说明新生儿的身体没有从血液中获得足够的氧气。这种情况还可能伴有急促、费力的呼吸声或呼噜声。这时，必须立即就医。

**排便**。新生儿出生后，医护人员会密切关注他首次排尿和排便的情况，确保他的排泄功能没有问题。新生儿的首次排便可能会推迟到出生24小时后货更晚。第一次或第二次排出的大便是黑色或深绿色的，非常粘稠，这是在他出生前就积蓄在其肠道里的胎便。如果新生儿出生后48小时内没有排出胎便，医生会进一步检查他的肠道是否存在问题。偶尔，新生儿的大便会带一点血，如果这种情况发生在出生后的最初几天，就意味着他的肛门因排便产生了小裂口。一般来说，这不会造成太大的伤害。但就算如此，你也要将孩子便血的情况告知儿科医生以便确认原因，因为还有其他因素可能造成出血，需要进一步检查并治疗。

**呛咳**。新生儿如果吃奶或喝水太快，就有可能呛咳。但是，这种呛咳一般不会持续太久，他一旦熟悉了喂养模式，就不会再呛咳。这种呛咳与母乳的喷射速度有关，尤其是刚开始下奶的时候。如果新生儿持续呛咳，吃奶时总是被呛住，或者吐奶过多，你就需要向儿科医生咨询了。这些表现可能提示他的肺部或消化道存在问题。

**过度哭闹**。所有新生儿都爱哭，且没有明显的原因。如果你确定已经给孩子喂过奶，也拍过嗝了，同时他的尿布也是干爽的，那么此时最好的办法可能就是抱着他，跟他说说话，为他唱唱歌，直到他停止哭闹。在孩子这么小的时候，你对他关心再多也不会宠坏他。如果上述努力都无效，你可以用一条婴儿抱毯把他包起来，可以向儿科医生咨询。孩子哭个不停，对任何父母来说都是压力。如果你发现自己开始感到极度沮丧，请将孩子交给其他看护者，或者将他放在婴儿床里或其他安全的地方，花点时间让自己平静下来。

你会逐渐习惯孩子正常哭闹的方式。如果孩子发出奇怪的哭声，比如源于疼痛的尖叫声，或者哭闹的时长超出正常范围，就有可能提示他生病了。这时，你需要向儿科医生咨询。

**产钳留下的印记**。如果产科医生在你分娩的过程中使用了产钳，金属产钳可能会在受到压迫的皮肤上（通常在新生儿的面部和头部皮肤上）留下红色的印记，有时甚至会造成擦伤。这种印记往往在几天内消失。有时候，当产钳轻微地伤到皮肤下的结缔组织时，新生儿的受伤部位就有可能出现突出于体表的、扁平而硬实的包块，这种包块通常会在2个月内消失。

**黄疸**。很多健康的新生儿皮肤会呈淡黄色，这被称为“黄疸”。这是由新生儿血液中一种叫作“胆红素”的化学物质蓄积引起的。轻微地黄疸是无害的，但如果胆红素水平持续升高且没有及时治疗，黄疸可能导致新生儿的大脑受损。黄疸更容易发生在母乳喂养的新生儿，尤其是吃奶情况不好的新生儿身上。进行母乳喂养的母亲应该注意，每天都应该给新生儿喂8~12次奶，这样不但可以保证足够的产奶量，还可以将新生儿体内的胆红素控制在较低的水平。

黄疸最初出现在面部，然后是胸部和腹部，最后可能是四肢。同时，新生儿的眼白会变黄。如今，在新生儿出生后24小时内，医院都会使用无痛手持式检测仪进行黄疸筛查。根据新生儿的皮肤颜色、出生天数以及其他因素，儿科医生如果怀疑他可能出现黄疸，就会为他进行皮肤或血液检测以辅助诊断。如果新生儿在出生后24小时之内出现黄疸，医生会为孩子抽血以便进行更为精确的胆红素测定。出院回家后，如果你发现孩子身上的黄疸忽然加重，喂养进行得不理想，或排尿次数减少，请和儿科医生联系。在新生儿出生后3~5天应有医生或护士检查他是否出现黄疸，因为这个阶段往往是胆红素水平最高的时候。新生儿如果在出生后不到72小时就出院了，就需要在出院后2天内再去医院接受检查。某些新生儿甚至需要更早去医院，比如：

- 出院前胆红素水平较高的新生儿。
- 较早出生的新生儿（比预产期提前2周以上出生）。
- 喂养进行得不理想的新生儿。
- 出生后24小时内出现黄疸的新生儿。
- 分娩过程中出现头皮下受伤或出血的新生儿。
- 亲生父母或哥哥姐姐曾经患胆红素血症并接受过治疗的新生儿。

胆红素升高可以用光疗法（也称“光照疗法”）进行治疗。新生儿会戴上遮光眼罩，被脱光衣服放到光疗箱中或用毯式光疗仪包裹着接受特殊光线的照射。这种治疗在家里和医院里都可以进行，可以预发黄疸对新生儿产生的负面影响。母乳喂养的新生儿的黄疸可能持续2~3周；配方奶喂养的新生儿的黄疸可能只需要2周就会彻底消失。

**嗜睡**。新生儿大部分时间都在睡觉。只要他每隔几小时都会醒来大吃一顿，看起来非常满足，就是完全正常的————哪怕他一天中的其他时间都在睡。但如果他很少保持清醒，或者不会因为饿了而醒过来，或者看起来非常疲惫且对进食毫无兴趣，你就应该向儿科医生咨询。这种嗜睡，特别是新生儿突然变得嗜睡，可能意味着他患上了严重的疾病。

**呼吸窘迫**。新生儿出生几个小时后才能形成正常的呼吸节奏，之后，他就不应该存在呼吸方面的困难了。如果新生儿看起来以一种奇怪的方式呼吸，很可能是因为他的呼吸道被堵塞了。你可以用生理盐水滴鼻液给他滴鼻，然后用吸鼻器或其他设备为他清晰鼻腔，将粘液冲洗出来。生理盐水滴鼻液是非处方药，可以在药店买到。

然而，如果新生儿出现了下述任何一种值得注意的体征，你就应该立即通知儿科医生。

- 呼吸频率加快（每分钟超过60次）。虽然新生儿正常的呼吸频率高于成年人，但是异常的呼吸加快值得注意。
- 胸部肌肉凹陷（每次吸气的时候肋间肌凹陷，使得肋骨突出）。
- 鼻翼扩张。
- 呼吸时发出呼噜声。
- 嘴唇或其他部位的皮肤持续发绀。

**脐带**。你需要保持新生儿的脐带残端干净清爽，直到它自行萎缩脱落。不需要在脐带残端上擦酒精，只需要让它保持清洁。

另外，一定要将盖肚脐的纸尿裤边缘折到肚脐之下，防止尿湿的止尿裤浸湿脐带残端。当脐带残端脱落时，你可能注意到纸尿裤上出现几滴血迹，这是正常的，但如果脐带出血不止，你就应立即带孩子就医。如果脐带残端发生了感染，则需要药物治疗。虽然这种感染并不常见，但如果孩子出现下面这些体征，你就应该带他就医。

- 脐带处出现黄色、有异味的分泌物。
- 脐带基部周围的皮肤发红。
- 碰到孩子脐带或周围皮肤时，孩子还是哭闹。

一般在新生儿出生后3周，脐带残端会干燥、脱落。如果3周以后脐带残端还没有脱落，则需要去看医生。

**脐肉芽肿**。有时候，脐带残端没有完全干燥就脱落，在新生儿的肚脐上留下肉芽肿或一小块红色或褐色的痂块。这种肉芽肿还有可能流出淡黄色的液体。这种情况一般一周内消失，如果没有，儿科医生可能会帮新生儿干燥或烧掉（化学灼烧并消毒）这部分肉芽肿。

**脐疝**。新生儿如果在哭闹的时候肚脐部位膨出，可能是患上了脐疝————新生儿腹壁肌肉上存在一个小洞（裂隙），当腹内压增大时（如哭闹时），腹腔内的组织就有可能从这个小洞往外突出。这不是一种严重的疾病，往往在孩子12~18个月大前就能自愈。少数孩子在4~5岁时还没有自愈，在这种情况下，孩子就需要接受手术。不要用胶带或硬币把肚脐堵起来，这种做法不仅无法解决问题，还会引发皮疹。
